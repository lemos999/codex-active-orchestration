# Source Fidelity

Prefer primary sources, official documentation, release notes, standards, repository files, papers, or direct datasets. Use secondary sources for orientation only unless the task is about secondary coverage. For current facts, verify with fresh lookup when the claim could plausibly have changed.

For every important claim, state whether it is verified, inferred, or unresolved. Keep dates concrete. Avoid long quotations; summarize and cite the source locator. If sources disagree, compare scope, date, authority, and evidence instead of averaging them.

When research affects an implementation or artifact, write the stable conclusion into an evidence ledger row and cite that row in downstream work.
